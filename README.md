# Unemployment Analysis in India

Data analysis project exploring unemployment trends across Indian states with a
focus on the COVID-19 lockdown impact.

## Overview
- Data cleaning, missing value imputation, and feature exploration
- Urban vs Rural unemployment comparison
- Time-series trend analysis (2019–2020)
- COVID-19 before vs after lockdown quantification (Apr 2020)
- Employment loss estimation in millions per state

## Key Findings
- National unemployment rate more than **doubled** during lockdown (peak >20%)
- Urban areas were hit significantly harder than rural
- States like Tripura and Haryana showed high pre-COVID unemployment (structural issues)

## Libraries
`pandas` `numpy` `matplotlib` `seaborn`

## Dataset
[Unemployment in India - Kaggle](https://www.kaggle.com/datasets/gokulrajkmv/unemployment-in-india)

## How to Run

2. Install requirements: `pip install pandas numpy matplotlib seaborn`
3. Open `main.ipynb` in Jupyter Notebook
